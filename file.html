  5. Once the instance starts, click JupyterLab.

4Ô∏è‚É£ Install PHREEQC in JupyterLab

Compile PHREEQC from Source

Open a terminal in JupyterLab and run the following commands:

cd ~ wget https://water.usgs.gov/water-resources/software/PHREEQC/phreeqc-3.8.6-17100.tar.gz tar -xvzf phreeqc-3.8.6-17100.tar.gz cd phreeqc-3.8.6-17100/src make sudo make install

Verify installation:

phreeqc --version

5Ô∏è‚É£ Run PHREEQC Simulations

Method 1: Using Python

phreeqc.load_database("/usr/share/phreeqc/database/phreeqc.dat") input_str = """ SOLUTION 1 pH 7.0 units mol/kgw Na 1.0 Cl 1.0 END """ phreeqc.run_string(input_str) print(phreeqc.get_selected_output())

Method 2: Using Command Line

Create a PHREEQC input file in JupyterLab:

SOLUTION 1 pH 7.0 units mol/kgw Na 1.0 Cl 1.0 END

Run PHREEQC in the terminal:

phreeqc example.phr output.txt /usr/share/phreeqc/database/phreeqc.dat cat output.txt
